NestJS SaaS Kit is a production-ready backend boilerplate that saves you weeks of development time. Built with NestJS 11, it comes with authentication, role-based access control, and Stripe subscription billing pre-integrated.

### What's Included

✅ **JWT Authentication** — Register, login, refresh token rotation, bcrypt password hashing
✅ **Role-Based Access Control** — USER, ADMIN, CREATOR roles with decorator-based guards
✅ **Stripe Payments** — Checkout sessions, subscription lifecycle, webhook event handling
✅ **Subscription Management** — Create checkout, view current plan, cancel at period end, change plan mid-cycle
✅ **Stripe Webhook Handling** — Signature verification in production, mock mode for local development
✅ **93%+ Test Coverage** — 82 unit tests + 34 E2E tests, all passing
✅ **API Security** — All endpoints protected by JWT, webhook secured by Stripe signature

### Tech Stack

| Layer | Technology |
|-------|-----------|
| Framework | NestJS 11 |
| Language | TypeScript 5 |
| Database | PostgreSQL (Prisma 7 ORM) |
| Auth | Passport.js (JWT + Local) |
| Payments | Stripe SDK v22 |
| Testing | Jest + Supertest |
